March 27, 2026. Day 1

The Question

What if AI could help us find HD treatments faster? 41,000 patients, 200K at risk, and the first treatment to slow the disease by 75% is stuck in regulatory limbo. We're data scientists, not doctors. but we can build tools.

search
Day 1. Research Phase

Mapping the Landscape

Scanned X, Reddit, YouTube, HN, PubMed, and the web for everything HD-related. Found 7+ active therapies, discovered nobody is running autonomous AI agents on HD research, and identified the key players: uniQure, Novartis, Unlearn, SOM Biotech.

Key finding: Novartis used generative AI to design 15M compounds. Unlearn built digital twins from 30K patients. SOM Biotech's AI discovered a drug now in Phase II. But nobody was doing open-source autonomous research.

Built the Research Platform

In one session: literature agent (PubMed), drug repurposing scanner (16 HD targets), trial tracker (ClinicalTrials.gov), and autoresearch loop (Karpathy-style overnight experiments). All open source, all running on an NVIDIA Jetson AGX Orin.

science
Day 1. First Experiment

Experiment #1: 22 Papers Analyzed

Fed 22 HD papers to Llama 3.1 8B on Jetson. The AI extracted targets, compounds, and key findings from each, then generated 5 drug repurposing hypotheses. Top scorer: Tocilizumab (IL-6 blocker) for HD neuroinflammation at 80/100.

Read the full report arrow_forward
What worked: LLM found cross-paper patterns (TDP-43/HD connection, post-translational modifications). What didn't: Suggested Riluzole, already tested with modest results. Honest about both.
database
Day 1. Paper Scout

46 Papers in the Corpus

Ran the Paper Scout agent across 12 different search queries. Built a growing corpus of analyzed HD research. each paper tagged with category, targets, compounds, findings, and repurposing potential.

Day 1. Autonomous Agents

Built 4 Research Agents

Paper Scout (discovers new papers every 6h), Hypothesis Refiner (re-scores drug candidates against new evidence), Social Watcher (monitors X, Reddit, YouTube, HN for HD chatter), and Digest Writer (compiles weekly research summaries). All running on Jetson + Mac via Ollama.

chat
Day 1. Ask HD Research

RAG Chatbot with Guardrails

Built a chatbot grounded in our corpus. answers questions by citing real papers, not hallucinating. Runs on NVIDIA NIM via Vercel serverless. Added medical advice guardrails: input filtering catches 30+ patterns before the LLM even runs, system prompt enforces hard rules, every response disclaims.

Day 1. Launched

Site Live + Daily Auto-Refresh

Published everything: dashboard with live data from 4 sources, experiment reports, chatbot (NVIDIA NIM + RAG over corpus), resources, privacy policy, guide. in 21 languages including 10 Indian languages. GitHub Actions refreshes data daily.
